What is Sparklyr & How To Install It

In order to install sparklyr and Apache Spark, you will need to run the following code. You will have to use the devtools package to retrieve sparklyr from the RStudio sparklyr repository on GitHub. You will then use the function spark_install() and provide a character string of the version you wish to install. 

  # Load sparklyr & dplyr
  devtools::install_github("rstudio/sparklyr")
  library(sparklyr)
  library(dplyr)

  # Install spark
  spark_install(version = "1.6.1")
  # Connecting to spark using spark_connect, on a local connection.
  sc <- spark_connect(master = "local")
